# 回收

## 本意

Doorman 曾有两天外出，布置要求做高考分类汇编。返回之后对分类汇编做了检阅，发现有很多人没有做

Doorman: "大家的高考分类汇编已经被我回→收→了。很多普通班的同学都想要啊大家注意。"

Doorman: "你们没有时→间→→啊，你们要吃饭，要睡觉，要上厕所，要维持你们体内的生态系统正常运转"

## 派生用法

可以用来 "回收" 所有东西

???+ note "例子1"
    第三节晚自习的时间是 20:40 ~ 21:30
    
    Daisy 曾在一次英语晚自习公然布置了 20:35 ~ 21:35 的任务

    或曰: "你们的 公共自习 / 课间 已经被我回收了"
